# 🧮 المنصة التفاعلية للحساب العمودي | Plateforme Interactive de Calcul Vertical

هي منصة تفاعلية مبتكرة مخصصة لمرافقة تلاميذ المرحلة الابتدائية في تعلم وإتقان العمليات الحسابية العمودية. توفر المنصة تجربة تعليمية ممتعة ومبسطة تشرح خطوات الحل بدقة.

---

## 🚀 المميزات الرئيسية (Caractéristiques)

* **العمليات الحسابية الأساسية**: دعم كامل للعمليات العمودية الأربع:
  * الجمع ➕
  * الطرح ➖
  * الضرب ✖️
  * القسمة الإقليدية ➗
* **واجهة بصرية جذابة**: تحتوي على رموز متحركة ومضيئة تجذب انتباه الأطفال وتسهل الفهم.
* **شرح تفصيلي**: توضيح خطوات الحل بدقة خطوة بخطوة بالتوازي مع العملية.
* **وحدة مخصصة للوقت**: جزء خاص بحسابات الوقت والزمن العمودية (الساعات، الدقائق، والثواني).

---

## 🛠️ التقنيات المستخدمة (Technologies)

تم بناء هذا المشروع باستخدام التقنيات الأساسية للويب لضمان السرعة والخفة:

* **HTML5**: لبناء الهيكل الأساسي للمنصة.
* **CSS3**: لتصميم الواجهة وإضافة المؤثرات البصرية والرموز المضيئة والمتحركة.
* **JavaScript**: للتحكم في منطق العمليات الحسابية وتوليد خطوات الحل التفاعلية.

---

## 💻 كيف يعمل المشروع؟ (Comment ça fonctionne)

1. قم بتحميل أو استنساخ المستودع (Clone repository).
2. افتح ملف `index.html` في أي متصفح ويب.
3. اختر العملية الحسابية وابدأ التعلم التفاعلي!
